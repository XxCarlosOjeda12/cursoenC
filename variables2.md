## Entradas :


```
a = 5
b = 18
c = 15
d = 25
```
## Proceso :

```
a = a + 10 = 5 + 10 = 15
b = b + 5 - c = 18 + 5 - 15 = 8
c =  c + 4 + b = 15 + 4 + 8 = 27
d = d + b + a = 25 + 8 + 15 = 48
a = a + 1 = 15 + 1 = 16 
b = b + c = 8 + 27  = 35
c = b + c = 35 + 27 = 62
d = b + b = 35 + 35 = 70
```
